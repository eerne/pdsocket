Pd-Socket
=========

Opens vanilla Pd and connects on a socket with [netreceive].

Read more about Puredata aka Pd on [crca.ucsd.edu/~msp/software.html](http://crca.ucsd.edu/~msp/software.html)

How to use
----------

	python pd-socket.py

### Messages

	pd.send('Hello Pd!;')
